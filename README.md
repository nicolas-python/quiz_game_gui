# Python Vokabel Quiz (GUI)

Ein Quiz-Spiel mit tkinter, um Python-Vokabeln zu lernen und zu wiederholen.

## Funktionen
- Spieler erstellen und auswählen
- Score und Spieler wird in einer SQLite-Datenbank gespeichert
- Zufällige Fragenreihenfolge
- Richtige / falsche Antworten werden farbig angezeigt
- Weiter-Button
- Zürück-Button (alte fragen nochmal ankucken)
- Erklärungen zu jeder Frage nach der Antwort
- Spiel kann jederzeit ins Hauptmenü beendet werden
  
## Installation
1. Projekt herunterladen
  - Öffne die GitHub Seite
  - Klicke auf grünen Button „Code“
  - Klicke auf „Download ZIP“
  - ZIP Datei entpacken (Rechtsklick → „Alle extrahieren“)
  
2.Ordner öffnen
  - Öffne den entpackten Ordner
  - Du siehst einen Ordner wie: quiz_game_gui-main

3.Spiel Starten
   Im Ordner: quiz_game_gui-main
  - SHIFT + Rechtsklick
  - PowerShell öffnen ("In Terminal Öffnen" klicken )
  - dann direkt: py main.py
 
## Steuerung
- Maus: Antworten anklicken
- Enter: nächste Frage
- "Weiter": nächste Frage
- "Beenden": zurück ins Hauptmenü

## Ziel des Projekts

Dieses Projekt dient sowohl zum Lernen von Python als auch zum Üben von Programmier-Vokabeln.

- Lernen und Wiederholen von Python-Vokabeln
- Verständnis von Grundlagen durch kurze Erklärungen
- Entwicklung eines eigenen Lern-Tools

- Python Grundlagen
- GUI Programmierung mit tkinter
- Arbeiten mit Datenbanken (SQLite)
- Logik und Spielmechaniken
